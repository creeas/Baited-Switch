# Baited Switch

This is a GitHub repository for an [NCEAS MORPHO](https://www.nceas.ucsb.edu/morpho) for the [Baited Switch Project](https://www.nceas.ucsb.edu/workinggroups/morpho-baited-switch-global-trade-driving-unsustainable-fisheries) Analysis.



### Repository Structure

This repository contains all the code associated with the project and figures produced by this code, and is organized into sub-modules. Due to space/version restrictions, project data are kept in a google drive outside of the repository and will be made available upon project completion and publication.
Data used includes landings information from the [Global Shark Meat Project](https://github.com/mamacneil/Global_Shark_Meat) and the [ARTIS](https://artisdata.weebly.com/) trade database.

This repository contain sub-modules for three sub-projects:

- [Fish n Chips](https://github.com/creeas/NCEAS-FishNChips)
- [Skates](https://github.com/NickDulvy/skate-meat-trade)
- [Silky Sharks]()



### Repository Best Practices

We ask that contributors adhere to the following best practices when working in the repository:

- Use lower case for all folders and file names
- Use underscores for spaces (don't use hyphens or spaces)
- For analyses with a set sript workflow please begin file names with two digit numbers (e.g. 01_, 02_, etc.)
